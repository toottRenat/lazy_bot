Доступные опции файла demon:
- поиск в гугле
- поиск видео в ютубе
- запуск программ
- запись в файл
- отправка сообщения в вк
- звонок по скайпу
Все опции необходимо настроить под себя, чтобы удобно ими пользоваться. Файл demon_settings как раз для этого и создан.

Также стоит учесть, что для нормального функционирования необходима программа AVbin
