1 опция активируется словом поиск
после слова поиск
следует ответ от программы
и необходимо называть слова или предложения которые нужно найти в интернете

2 опция активируются слову(словом) запуск
необходимо назвать программу которую нужно запустить
следует учесть что программа должна находиться в Past(PATH)
иначе программа просто-напросто не будет запущена

3 выпуск(опция) активируются словом запись
после этого необходимо называть название файла
и говорить те предложения которые будут записаны в файл
если файл уже существует с таким названием то данные будут записаны в конец файла
(в)противном случае будет создан файл
в конце записи необходимо
сказать фразу закончить запись
в случае(в этом случае) данные будут сохранены
а если этого не сказать и просто Закрыть программу то данные сохранены не будут и запись (прервется)

Также стоит учесть, что для нормального функционирования необходима программа AVbin
